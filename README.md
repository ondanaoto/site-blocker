# Time-based Site Blocker

指定したサイトを、設定した時間帯にだけブロックする Chrome 拡張機能。
ブロックを「緩める」変更 (ドメイン削除・時間帯短縮など) は、設定した「編集窓」の時間内でしかできません。

## インストール

### 1. クローン

```sh
git clone git@github.com:ondanaoto/site-blocker.git
```

### 2. Chrome に読み込む

1. Chrome で `chrome://extensions/` を開く
2. 右上の **デベロッパーモード** を ON にする
3. **「パッケージ化されていない拡張機能を読み込む」** をクリック
4. クローンしたディレクトリ (`site-blocker/`) を選択

### 3. シークレットモードで有効化

シークレットウィンドウでもブロックを効かせるには、ユーザー側で明示的に許可する必要があります。

1. `chrome://extensions/` で **「Time-based Site Blocker」** の **「詳細」** を開く
2. **「シークレット モードでの実行を許可する」** を ON にする

## 使い方

- ツールバーのアイコン → **「オプション」** から、ブロック対象のドメインと時間帯を設定します。
- 1 つのルールに複数ドメイン / 複数時間帯を登録できます。
- 終了時刻が開始時刻より前のときは、日をまたぐ範囲として扱います (例: `22:00 → 02:00`)。
- 「編集窓」の時間外は、ルールを **厳しくする変更** (ブロック対象の追加・時間帯延長など) しかできません。
